# ModifyEXIF
Modify your photos EXIF, if you hate the wrong date.

# Requirements
+ pyexiv2  
+ os,re,time  
+ win32file  
+ pywintypes  

# How to use
1.Change `pic_dir` in `if __name__ == "__main__":`.  
2.Then just `python test.py'.
